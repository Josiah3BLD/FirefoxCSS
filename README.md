# FirefoxCSS
A custom Firefox skin

![Preview Image](/Preview/Showcase.jpg)

Screenshot taken with [Matte Black (Sky Blue)](https://addons.mozilla.org/en-GB/firefox/addon/matte-black-sky-blue/) addon by Elijah Lopez

### Installation
1. Download FirefoxCSS from releases
2. Create a new Firefox profile in about:profiles, but **do not** open it
3. Move user.js and the chrome folder into the new Firefox profile folder
4. Startup the new Firefox profile
5. Open user.js in Notepad and remove the user_prefs after the gap as instricted by the comment
6. Save and close user.js
7. Customise toolbar inside Firefox as wanted
